FlexBox Notes

CSS: Flex-Container Properties

display:

defines a flex container

display: flex sets an HTML element as a block level flex container which takes the full width of its parent container
inline-flex sets an HTML element as an inline flex container which takes only the required space for the content

flex-direction:

establishes the main-axis, defines the direction flex items are placed in the flex container

flex-direction: row (default) left to right
row-reverse right to left
column same as 'row' but top to bottom
column-reverse same as 'row-reverse' but bottom to top

flex-wrap:

determines whether flex-items wrap or try to fit on one line

flex-wrap: nowrap (defult) all flex items on one line
wrap flex items wrap onto multiple lines, from top to bottom
wrap-reverse flex items wrap onto multiple lines from bottom to top
